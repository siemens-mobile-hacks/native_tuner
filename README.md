# NativeTuner – Native ELF FM Radio for Siemens Mobile Phones

NativeTuner is a native ELF application that replaces the built‑in Java FM radio on Siemens mobile phones. It offers a faster, more responsive interface and extended functionality.

## Building
The SDK must be located at `../sdk` relative to the project root.
```bash
mkdir build && cd build
cmake ..
make
```
